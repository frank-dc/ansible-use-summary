# TCP 和 UDP 的比较
| 区别     |               UDP               |               TCP               |
|--------|:-------------------------------:|:-------------------------------:|
| 是否可靠   |               不可靠               |               可靠                |
| 是否面向连接 |                否                |                是                |
 | 连接对象个数 |        一对一、一对多、多对一、多对多通信        |              一对一通信              |
| 传输方式   |              面向报文               |              面向字节流              |
| 首部开销   |              首部8字节              |         首部至少20字节，最大60字节         |
| 适用场景   | 效率要求相对高，准确性要求相对低（在线聊天、视频、即时通信等） | 效率要求相对低，准确性要求相对高（文件传输、邮件、远程登录等） |
